# terminal終端機
## 印出工作目錄
```shell
pwd
```
## 列出資料夾內容(list)
```shell
ls
ls -lh #列出詳細資料
```
## change directory
```shell
cd 資料夾
```
## 回到上一層資料夾
```shell
cd ..
```
## 回到家目錄
```shell
cd ~
```
## mkdir 建立一個目錄(資料夾)
## mkdir {1..10} 新增資料夾 1~10
## mkdir {a..z}{1..100} 新增資料夾a to z且每個資料夾內都含有1 to 100子資料夾
```shell
mkdir 資料夾
```
## tab鍵 自動補完後面的字
## 複製貼上(Linux) ctrl+shift+C/V
## rmdir 刪除一個目錄(要小心！！！）
## 複製
```shell
cp 檔名 資料夾
```
## 移動
```shell
mv 檔名 資料夾
```
## 看文字檔
```shell
less 文字檔
```
## 日曆
```shell
cal
cal 2021
```
## 清空畫面
```shell
clear
```
## 文字編輯器
```shell
nano 文字檔
```
## superuser do 最高權限
```shell
sudo 指令
```
## manual 查指令
```shell
man 指令
```
### touch:creates a new file
```shell
touch grocery-list.txt
```
